# VI. Winning the Game

## 34. The End of the Game

**34.1** The game ends when either of the following conditions is met:

**A.** At least one player moves his marker onto a **finish square** on the A.S.T. The move into the finish square counts as the last step. The player who first reaches a finish square does **not necessarily** win.

**B.** A **predetermined time limit** is reached.

**34.2** Because reaching the end of the A.S.T. takes a long time, games will often end at a time limit. The time limit **must be set before the game starts**. Players must **complete the final turn** before determining the winner.

---

## 35. Victory Determination

**35.1** The winner is determined by adding the value of all of the following:

| Component | Points |
|-----------|--------|
| **A. Civilization cards** | Total face value of all held cards |
| **B. Commodity cards** | Total value of sets (28.51) plus face value of individual cards |
| **C. Treasury** | One point per token in treasury |
| **D. A.S.T. position** | **100 points** per space on the A.S.T. (counting from the start arrow) |
| **E. Cities** | **50 points** per city on the board |

**35.2** The player with the **highest point total** wins. This will not necessarily be the player with the most valuable civilization cards or the player furthest along the A.S.T., although both are important sources of points.

If two or more players tie, the player **furthest along the A.S.T.** wins the tiebreaker.
