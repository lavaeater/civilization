# Cilivazation - Advance!!!

The best board game of all time is Advanced Civilization. A dear friend of mine was the first (unofficial) swedish champ of the game, and this is my tribute to him. If I make even one thing happen during my lifetime that is done, and it is this, I can die happy.

Based off the Bevy Game Template.
